# RayHub AI LMS

RayHub is an AI-powered Learning Management System designed to help Kasneb(first focusing on CPA) students track progress, practice past papers, and receive intelligent study recommendations.

## Features

- FastAPI backend
- SQLAlchemy ORM
- JWT Authentication
- CPA course tracking
- Quiz and past paper practice
- AI study recommendations (coming soon)

## Tech Stack

FastAPI  
SQLAlchemy  
SQLite / PostgreSQL  
JWT Authentication  
Python

## Vision

To build an intelligent LMS that helps CPA students prepare effectively using AI-driven insights.
